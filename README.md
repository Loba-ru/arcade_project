# arcade_project "Monster Chase"

Платформер на Python с использованием библиотеки Arcade.

## Общие сведения

Проект построен на классическом паттерне `State` (Состояние) для управления экранами и паттерне `MVC` (Model-View-Controller) в упрощенном виде для самой игры.

1. `StateManager` (в `states.py`) — это «Контроллер» для экранов. Он управляет тем, какой экран сейчас активен (`StartView`, `MenuView`, `GameplayView` или `ResultView`).
2. `MyGame` (в `game_engine.py`) — это «Модель» и главный менеджер. Он хранит глобальное состояние (монеты, жизни) и управляет логикой переключения уровней.
3. `BaseLevel` (в `levels.py`) — это «Вид» (View) и локальный «Контроллер» для одного уровня. Он отвечает за отрисовку, физику и управление персонажем внутри уровня.
