# arcade_project "Monster Chase"

Платформер на Python с использованием библиотеки Arcade.

## Текущий статус

✅ **Первый пробный шаблон окна**

- Игрок (девочка-авантюристка) может ходить влево/вправо и прыгать
- Физика (гравитация, платформы)
- Земля прижата к низу окна
- Интерфейс: сердечки (жизни), счёт монет (заглушка)
- Камера: два слоя (мир и GUI)
- Мёртвая зона камеры и плавное следование за игроком

## Планируемые состояния

Вторым этапом добавим четыре состояния игры:
- Стартовое окно: Меню, кнопка "Start", возможно, лучший результат 
- Игра: Основной геймплей: сбор монет, уклонение от врагов
- Пауза: Остановка игры, возобновление по пробелу или Escape
- Результат: Экран Game Over с финальным счётом и кнопкой рестарта
